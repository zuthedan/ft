The following packages are currently available:  
	* ui
	* ui.content
	* ui.outline
	* ui.util
	

Compiling (MacOS / Linux)

Navigate to the src  folder of the project and execute the following:

compile:
javac -cp .:lib/* de/btu/sst/evs/blatt14/ui/content/*java de/btu/sst/evs/blatt14/ui/content/*java de/btu/sst/evs/blatt14/ui/*java

execute:
java -cp .:lib/opencsv-3.8.jar de.btu.sst.evs.blatt14.ui.BerlinAirApp
